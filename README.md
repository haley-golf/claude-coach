# claude-coach
**A self-contained, interactive 7-day course for getting the most out of Claude. Covers prompt engineering, Projects, web search, Claude Code, Cowork, MCP connectors, and advanced patterns — 15–30 minutes a day, with expandable lessons, hands-on practice, and progress tracking. Single HTML file, no dependencies.**

| Day | Topic | Focus |
|-----|-------|-------|
| 1 | How Claude Thinks | Mental model, structured prompting, examples |
| 2 | Projects | Persistent workspaces, instructions, file uploads |
| 3 | Claude.ai Deep Dive | Artifacts, image/PDF input, memory settings |
| 4 | Web Search & Research | Live search, Deep Research, multi-step workflows |
| 5 | Claude Code | Terminal-based AI, CLAUDE.md, agentic coding |
| 6 | Cowork & Ecosystem | Desktop automation, integrations, MCP layer |
| 7 | Advanced Patterns | Meta-prompting, roles, long-conversation hygiene |

Progress is saved in localStorage — your browser's built-in local storage. This means:  
  ✅ Progress persists across browser sessions on the same device  
  ✅ No account or login needed  
  ❌ Progress doesn't sync across devices (your phone and laptop have separate localStorage)  
  ❌ If you clear your browser data/cookies, progress resets  
  ❌ If multiple people use the same hosted URL, their progress is stored locally on their own device — they don't see each other's progress
